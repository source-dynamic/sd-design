# List列表
通用列表

## 概述
最基础的列表展示，可承载文字、列表、图片、段落，常用于后台数据展示页面。

## 滚动加载无限长列表
组件提供了虚拟列表功能，可实现滚动加载无限长列表，能够提高数据量大时候长列表的性能
